Important FAQ

What is the document object model(DOM)?

The dom is a programming interface for web document. It represents the structure of an html as a tree of object that can be accessed ,manipulated and modified using programming languages like javaScript.

How do you select an element from dom?

To select an element in the dom we use various method and properties.by that method we select the id or class of the element an manipulate it as required.

What is event delegation in the context of Dom,and why is it useful?

Event delegation is a technic in dom where a single event listener is added to a parent element rather than attaching separate event listener to multiple child element. this works because events triggered by child elements bubble up through the dom to their parent element,allowing the parents to listen for those events on behalf of the children.

How do you manipulate an elements attributes and styles using Dom?

To manipulate an element first we can select an element by using Dom methods and inject attributes with classlist and add different styles using css.
